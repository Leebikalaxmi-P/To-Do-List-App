# To-Do List App in Python

This is a simple **To-Do List application** developed using Python.
The program allows users to manage their daily tasks from the terminal by adding, viewing, and deleting tasks.

## 📌 Features

* Add new tasks
* View all tasks
* Delete a task
* Simple menu-driven program
* Runs in terminal / command prompt

## 📌 Language Used

* Python 3

## 📌 How it Works

The program displays a menu with options to:

1. Add task
2. View tasks
3. Delete task
4. Exit

User can choose an option and manage the to-do list easily.

## 📌 Example

Add Task → Buy groceries
Add Task → Complete assignment
View Tasks →

1. Buy groceries
2. Complete assignment
